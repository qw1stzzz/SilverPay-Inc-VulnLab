# Security Policy

**⚠️ This is an INTENTIONALLY VULNERABLE lab** created only for educational purposes.

It contains 15 real security vulnerabilities (SQL Injection, XSS, SSRF, XXE, etc.).

**DO NOT**:
- Run this VM on a production network
- Expose it to the internet
- Use it for any illegal activity

Use only in isolated environment (NAT or Host-Only network in VirtualBox/VMware).

Created for students and penetration testing practice.
